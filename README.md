# Medae
A game based on the concept of no built-in content. Content is created by developers and uploaded to the server, which then distributes resources and data to the client.

## Used external libraries (all files in the Library and Include directories)
https://github.com/glfw/glfw
https://github.com/Dav1dde/glad
https://github.com/g-truc/glm
https://github.com/nothings/stb
